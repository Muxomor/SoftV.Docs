## MVVM 模式

在本教程中，您将学习更多关于使用 Model-View-ViewModel (MVVM) 模式的知识。在编写图形用户界面应用程序时，通常会使用 MVVM 模式，并且在大多数情况下，Avalonia UI 应用程序都推荐使用 MVVM。

:::info
要了解有关在 Avalonia UI 中使用 MVVM 模式以及何时适用该模式的一些背景概念，请参阅：[这里](../../concepts/the-mvvm-pattern/).
:::
